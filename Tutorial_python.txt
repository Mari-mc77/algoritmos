O QUE É PYTHON
Python é uma linguagem de programação de alto nível, interpretada e multiparadigma, criada por Guido van Rossum em 1991. Sua sintaxe é fácil de ler e escrever, tornando-a ideal para iniciantes e profissionais. Python suporta programação orientada a objetos, funcional e imperativa. É amplamente utilizado em uma variedade de domínios, como desenvolvimento web, análise de dados, inteligência artificial e automação de sistemas, devido à sua simplicidade, flexibilidade e vasta coleção de bibliotecas.

QUEM CRIOU O PYTHON

Guido van Rossum criou o Python em 1991 na Holanda e liderou seu desenvolvimento até 2018, quando se aposentou como líder do projeto.


COMO É FEITO UM DOWNLOAD
Um download é feito quando você solicita que um arquivo ou dados sejam transferidos de um servidor para o seu dispositivo. Geralmente, você clica em um link ou botão de download em um site ou aplicativo e o arquivo é baixado para o armazenamento local do seu dispositivo.

PASSO A PASSO DA INSTALAÇÃO
Baixe o Instalador: Acesse o site oficial do Python em python.org e clique em "Downloads". Selecione a versão mais recente do Python para Windows e baixe o instalador executável
Execute o Instalador: Após o download ser concluído, clique no arquivo de instalação para iniciá-lo.
Configurações Iniciais: Na tela inicial do instalador, marque a opção "Add Python to PATH" (Adicionar Python ao PATH) e clique em "Customize installation" (Customizar instalação) se desejar alterar as opções padrão.
Seleção de Componentes: Na tela de seleção de componentes, deixe as opções padrão selecionadas a menos que tenha uma razão específica para alterá-las.
Seleção de Local de Instalação: Escolha o diretório onde deseja instalar o Python ou deixe o padrão e clique em "Next" (Próximo).
Instalação: Clique em "Install" (Instalar) para começar o processo de instalação. Aguarde enquanto o Python é instalado no seu computador.
Conclusão da Instalação: Após a instalação ser concluída, você verá uma mensagem informando que o Python foi instalado com sucesso.
Verificação da Instalação: Abra o prompt de comando e digite python --version para verificar se o Python foi instalado corretamente e para ver qual versão está instalada.
